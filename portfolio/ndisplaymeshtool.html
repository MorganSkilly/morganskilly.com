LED Wall Mesh Generator for Unreal nDisplay

A plugin for Unreal Engine to procedurally generate nDisplay projection meshes based on LED panel specifications. This tool streamlines the process of creating accurate projection surfaces for virtual production workflows using LED walls.

The plugin takes LED panel dimensions, resolutions, and configuration parameters to automatically generate the precise mesh geometry required for nDisplay systems. This eliminates the manual mesh creation process and ensures pixel-perfect alignment between the virtual content and physical LED panels.

Key Features

  • Procedural mesh generation based on LED panel specifications
  • Support for various LED panel configurations
  • Automatic UV mapping for proper texture projection
  • Integration with Unreal Engine's nDisplay system
  • Real-time preview and adjustment capabilities
