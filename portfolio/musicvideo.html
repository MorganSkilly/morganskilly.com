Virtual Production Music Video

I directed and edited a music video that utilised Unreal Engine and an LED wall inspired by Saint Motel's "A Good Song Never Dies". This project showcased the creative possibilities of virtual production technology in music video production.

The production involved real-time rendering of virtual environments using Unreal Engine, synchronized with LED wall displays to create immersive backgrounds that responded dynamically to the music and performance. This approach allowed for creative freedom and immediate visual feedback during the shoot.

Production Details

  • Real-time rendering using Unreal Engine
  • LED wall virtual production setup
  • Music-synchronized visual effects
  • Creative direction and editing
  • Integration of practical and virtual elements
