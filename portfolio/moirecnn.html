Detecting Spatial Aliasing (Moiré) Using a Convolutional Neural Network

Using PyTorch, I trained a CNN to binary classify 4k images containing spatial aliasing patterns (moiré effects). This project addresses a critical issue in virtual production where LED walls can create unwanted visual artifacts when captured by cameras.

The neural network was designed to automatically detect moiré patterns in real-time camera feeds, enabling immediate identification and correction of aliasing issues during virtual production shoots. This tool helps maintain image quality and reduces post-production work.

Technical Details

  • Built using PyTorch deep learning framework
  • Binary classification for moiré pattern detection
  • Trained on 4K resolution images
  • Real-time inference capabilities
  • Integration with virtual production pipelines
