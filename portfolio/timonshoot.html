Exploring the Previsualisation Pipeline for virtual production

This MSc project was led by Timon Holzer. As the on-set virtual production technical artist running the volume I was responsible for optimisation of Timon's project and ensuring it ran smoothly on the nDisplay cluster. The project involved many challanges including running a scene with a large number of Metahumans and frame remapping to improve the post-production pipeline. By gen-locking the camera and LED processors at 50hz and the Unreal scene at 25hz with a green/bluescreen every other frame, it is much easier to carry out post VFX work without the need to manually rotoscope footage.
